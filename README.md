# Bookmart-back

## Initial configuration
1. Create a python virtual environment
2. Get into your virtual environment shell
3. Go to the root folder
4. Execute the following commands:

        pipenv install


## Get the server running
```
python3 run.py
```


### Prerequisites

```
flask
pipenv

```
